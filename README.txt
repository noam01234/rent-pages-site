GitHub Pages publish folder for rent.html

Upload only the contents of this folder to the GitHub Pages repository.

Files:
- rent.html
- images\

Example public URLs after GitHub Pages is enabled:
https://YOUR-GITHUB-USER.github.io/YOUR-REPO/rent.html?id=demo1
https://YOUR-GITHUB-USER.github.io/YOUR-REPO/rent.html?id=demo2
https://YOUR-GITHUB-USER.github.io/YOUR-REPO/rent.html?id=demo3

Images:
Place apartment images inside the images folder.
Then update photo URLs in Supabase, for example:
images/demo-room-main.jpg
images/demo-room-2.jpg
images/demo-room-3.jpg

Important:
Do not upload tsm.html to the public rent-pages-site repository unless you intentionally want it public.
